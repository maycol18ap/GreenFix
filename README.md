# GreenFix
This is a decentralized financing system with smart contracts, and I'm developing an MVP to validate the business logic.

🚀 Guía Rápida para el Equipo
1. Configuración Inicial (Solo una vez)
Instalar Git.

Clonar el proyecto: git clone [https://github.com/tu-usuario/tu-repo.git](https://github.com/tu-usuario/tu-repo.git)

Abrir la carpeta en VS Code.

2. Cómo trabajar sin romper nada
Paso A: Crear tu propia rama (Antes de empezar a programar):

Bash
git checkout -b nombre-de-tu-tarea
Ejemplo: git checkout -b feat-login-backend

Paso B: Guardar y Subir cambios (Al terminar):

Bash
git add .
git commit -m "Explica brevemente qué hiciste"
git push origin nombre-de-tu-tarea
Paso C: El Pull Request:
Ve a la página del repositorio en GitHub y dale al botón verde "Compare & Pull Request". Espera a que el líder (yo) lo revise.
